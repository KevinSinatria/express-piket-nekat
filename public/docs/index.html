API Surat Izin Siswa (SISWA) (1.0.0)

Download OpenAPI specification:

API untuk mengelola surat izin keluar/masuk siswa. Implementasi ini menggunakan alur persetujuan dua tahap (Mapel dan Piket).

Endpoint untuk otorisasi dan manajemen sesi pengguna.

Membuat Surat Izin Siswa Baru

Guru Piket membuat dan mengajukan surat izin baru. Status awal: PENDING_MAPEL.

Authorizations:
bearerAuth
Request Body schema: application/json
student_nis
required
string

NIS siswa yang mengajukan izin.

reason
required
string

Alasan keluar/izin siswa.

hours_start
required
integer

Jam ke- mulai izin.

hours_end
required
null

Jam ke- akhir izin. Null jika izin sampai akhir hari/belum diketahui.

mapel_user_id
required
integer

ID pengguna Guru Mapel yang bertugas saat jam pelajaran.

Responses

Server Lokal/Development

http://localhost:5000/student-permits

Server Production

https://piket-nekat-api.vercel.app/student-permits

  • Payload
Content type
application/json
{
  • "student_nis": "232410465",

  • "hours_end": null,
  • "mapel_user_id": 62
}
